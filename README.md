# Maui UI Test with Appium
This repository contains a sample Maui UI test project that demonstrates how to use Appium for testing Maui applications. The project is set up to target .NET 10.0 and uses C# 14.0 features.

# Getting Started

## Maui Application
Maui is a simple sample app from Maui's standard templates. It includes basic UI elements such as buttons, labels, and entry fields that can be interacted with during testing.

## MSTest_Appium
MSTest_Appium is a test project that contains UI tests for the Maui application. It uses MSTest as the testing framework and Appium for automating interactions with the Maui app.
